## [1.0.0] - 2026-01-08

### Added
- Initial release of GamePay Hub
- Steam wallet top-up with 0% commission for 12 CIS countries
- Telegram Stars purchase with 2% commission
- Gift Games store with Steam games
- Gift Cards marketplace (Steam, PlayStation, Xbox, Nintendo, Apple)
- PayPalych payment integration (USDT via TON)
- GreenGamePay API integration for Steam services
- User authentication with Laravel Sanctum
- Order management system with status tracking
- Webhook handlers for payment notifications
- Docker development and production configurations
- CI/CD pipelines with GitHub Actions
- GitHub Pages deployment for demo preview
- Responsive dark gaming theme UI
- SEO optimization with Next.js SSR/SSG

### Security
- HTTPS enforcement
- CORS configuration
- Rate limiting on API endpoints
- Input validation and sanitization
- Secure webhook signature verification

## [0.1.0] - 2026-01-07

### Added
- Project scaffolding
- Basic frontend structure with Next.js 14
- Backend API with Laravel 11
- Database migrations for users and orders
- Docker Compose configurations
